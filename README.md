## Task 6: House Price Prediction

**Objective:** Predict house prices using property features.

**Dataset:** California Housing Dataset (sklearn built-in)

**Models Applied:** Linear Regression, Gradient Boosting

**Key Results:**
- No missing values in dataset
- MedInc has strongest correlation with house price
- Gradient Boosting outperforms Linear Regression
- Evaluated using MAE and RMSE

**Tools Used:** Python, Pandas, Scikit-learn, Seaborn, Matplotlib
